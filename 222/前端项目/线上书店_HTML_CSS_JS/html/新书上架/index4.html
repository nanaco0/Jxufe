海外馆藏中国文物精萃

寇勤 主编

内容简介

《海外馆藏中国文物精萃》为著名收藏家翁万戈先生拍摄的海外博物馆藏中国文物摄影作品精选集。书中不仅展示这些艺术珍品，并对它们进行了整体描述，包括入藏时间、价值认定、流转故事等，让更多国人了解中国文物流散情况及其目前的归宿。全书分上中下三卷，上卷为书画、敦煌遗画、壁画，中卷为青铜器、陶瓷器，下卷为玉器、金银器等。本书可作为国人海外文化之旅的导引指南，也是查询海外馆藏文物的权威工具书。值得文博界、收藏界、文物艺术品爱好者收藏阅读。
